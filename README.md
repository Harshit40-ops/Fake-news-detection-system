AI Fake News Detection System

An advanced Machine Learning + NLP based web application that detects whether a news article is Real or Fake. The project uses natural language processing techniques to analyze news text and predict its authenticity, helping users identify misinformation in the digital world.

Built with Python, Scikit-learn, and Streamlit, the application provides a modern interactive interface with visual analytics and AI-powered predictions.

🚀 Features

✅ AI Fake News Detection
Classifies news articles as Real or Fake using a trained machine learning model.

📊 Prediction Confidence Graph
Displays model confidence through a visual bar chart.

📈 Interactive Analytics Dashboard
Shows dataset insights like Fake vs Real news distribution.

🌍 Live News Integration
Fetches the latest headlines using News API.

🧠 NLP Text Processing
Includes text cleaning, tokenization, and vectorization for accurate predictions.

🌌 Modern AI Interface
Features 3D animated background, AI brain animation, and glassmorphism UI design.

⚡ Real-Time Detection
Users can paste any news headline or article to instantly check its authenticity.
